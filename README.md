# Notes App
Author: Artur Lekston <br>
Version: 1.0.0 <br>
License: MIT License

## About
Web application to managing your notes.

## Main features:
- easy and quick note creating with the Markdown editor
- instant synchronization with the server
- ability to assign notes to a category
- filtering notes by category

## Project Purpose
The primary project purpose is to get and deepen skills related to web software development.

## Requirements
To run the application, you need one of the modern, updated web browsers.

## Tech stock
**Frontend:** React, Material UI, Axios <br>
**Backend:** Symfony, API Platform, Lexik JWT Authentication, Gesdinet JWT Refresh-token <br>
**Additional technologies:** Docker, MariaDB, AI coding assistants <br>
